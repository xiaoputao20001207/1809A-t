### [git操作](https://jasonandjay.github.io/study/zh/standard/Start.html#git%E7%8E%AF%E5%A2%83)

+ 创建版本库
    + git clone url
    + git init
+ 查看状态
    + git status
+ 提交信息
    + init 
    + add 新增
    + update 更新
    + fix 修复
    + finish 完成
    + remove 删除
+ 撤销
    + 软撤销 git reset --soft []
    + 应撤销 git reset --hard []
+ 分支
    + 创建分支
    + 切换分支
    + 删除分支
    + 合并分支
+ 提交
    + 先拉去远程分支  pull
    + 在提交远程 push
